

Экзаменационный тест по столярному делу. 9 класс.

1. Какой инструмент используют для продольного пиления древесины?
   а) Лобзик
   б) Ножовка с мелким зубом
   в) Лучковая пила
   г) Шипорезная пила
2. Для чего предназначен столярный верстак?
   а) Для хранения инструментов
   б) Для крепкого удерживания заготовки при работе
   в) Для сушки древесины
   г) Для заточки инструментов
3. Основной частью рубанка является:
   а) Молоток
   б) Нож (резец)
   в) Стамеска
   г) Напильник

   б) ДСП (Древесно-стружечная плита)
   в) Брус
   г) Вагонка

   в) Вытереть кровь тряпкой

   г) Молоток

   б) Для выравнивания и сглаживания поверхности древесины
   в) Для сверления отверстий
   г) Для склеивания деталей
8. Струбцина – это инструмент для:

   г) Пиления

   в) Суперклей
   г) Клей для обуви

    г) Не работать

    б) Лобзик
    в) Рубанок
    г) Стамеска

    б) Для строгания древесины
    в) для пиления
    г) Для забивания гвоздей

    а) Киянка
    б) Отвертка
    в) Стамеска
    г) Клещи

    а) Цельного куска дерева
    б) Спрессованной стружки
    в) Склеенных тонких слоев шпона
    г) Пластика и опилок
15. Перед началом работы необходимо:
    а) Спрятать инструмент
    б) Надеть рабочую одежду и проверить исправность инструмента
    в) Быстро начать работу, чтобы успеть
    г) Поиграть с инструментом

    а) Молоток
    б) Струбцина

    г) Для шлифовки

    а) Гладко отшлифованная поверхность

    в) Использование клея
    г) Использование струбцины

    б) Знак "Наденьте перчатки"
    в) Знак "Наденьте защитные очки"
    г) Знак "Вымойте руки"

    в) Большая поперечная пила
    г) Рубанок

    а) Молотка

    в) Клея
    г) Струбцины

    а) Для заточки других инструментов
    б) Для сглаживания кромок и снятия небольшого слоя древесины
в) Для пиления досок
    г) Для разметки

    а) Тополь
    б) Осина
    в) Сосна, дуб, бук
    г) Ольха
